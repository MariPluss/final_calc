# final_calc
Калькулятор на Kivy с историей операций
Описание проекта:
Приложение-калькулятор, созданное с помощью Python и библиотеки Kivy.
Поддерживает базовые арифметические операции, а также дополнительные функции:

Возведение в квадрат (x²)

Квадратный корень (√)

Проценты (%)

Очистка (Сброс)

История последних 5 операций

Используемые методы и параметры:

eval() — для вычисления выражений;

math.sqrt() — для извлечения квадратного корня;

Popup и Label — для вывода истории операций;

ids.display.text — для отображения и очистки ввода.
